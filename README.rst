findx - an extended ``find`` command.
=====================================

Installation
------------

Package can be installed via ``easy_install`` or ``pip``.

* Create the binary distribution::

    python setup.py bdist

* Install via ``pip``::

    pip install .

or

* Install via ``easy_install``::

    easy_install .
