findx - an extended ``find`` command.
=====================================

Installation
------------

From PyPI, installation is the usual::

  pip install findx

From the source tree, install via::

  python setup.py install

Running the tests
-----------------

Invoke the tests via the ``Makefile``::

  make

Or manually via::

  python -m unittest discover
